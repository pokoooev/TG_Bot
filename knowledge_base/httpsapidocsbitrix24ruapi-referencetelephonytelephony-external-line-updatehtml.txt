## ОПИСАНИЕ
Scope: telephony Кто может выполнять метод: администратор Метод telephony.externalLine.update позволяет изменить название внешней линии. Параметр / Тип Описание NUMBER
string Номер внешней линии. NAME
string Название внешней линии. CRM_AUTO_CREATE
boolean Данный параметр отвечает за создание CRM-сущностей (сделка или лид, в зависимости от режима работы CRM) при исходящих звонках из интерфейса Битрикса (например, номеронабиратель в панели чата). Принимает значение Y|N. По умолчанию Y.
