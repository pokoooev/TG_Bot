## ОПИСАНИЕ
Scope: department Кто может выполнять метод: пользователь с правами на изменение структуры Метод department.delete удаляет отдел в структуре компании. Обязательные параметры отмечены * Название
тип Описание ID*
int Идентификатор отдела Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
boolean Результат удаления отдела в структуре компании time
time Информация о времени выполнения запроса HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки.
