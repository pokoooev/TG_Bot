## ОПИСАНИЕ
Scope: calendar Кто может выполнять метод: любой пользователь Метод получает список календарей. Обязательные параметры отмечены * Название
тип Описание type*
string Тип календаря: ownerId*
integer Идентификатор владельца календаря. Для типа календаря location параметр ownerId должен иметь значение 0 Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
array Массив календарей ID
string Идентификатор календаря NAME
string Название календаря GAPI_CALENDAR_ID
string Идентификатор синхронизации DESCRIPTION
string Описание календаря COLOR
string Цвет календаря TEXT_COLOR
string Цвет текста в календаре EXPORT
object Объект с параметрами экспорта календаря CAL_TYPE
string Тип календаря OWNER_ID
string Идентификатор владельца календаря. Для типа Календарь пользователя user поле содержит идентификатор пользователя. Для Календаря группы group — идентификатор группы CREATED_BY
string Идентификатор создателя календаря DATE_CREATE
datetime Дата создания календаря TIMESTAMP_X
datetime Дата изменения календаря CAL_DAV_CON
string Идентификатор синхронизации SYNC_TOKEN
string Идентификатор синхронизации PAGE_TOKEN
string Идентификатор синхронизации EXTERNAL_TYPE
string Тип провайдера для синхронизации ACCESS
object Объект данных доступа к календарю. Ключ объекта — идентификатор прав доступа. Получить название прав доступа можно методом access.name. Определить права доступа для текущего пользователя — методом user.access. Значение обьекта содержит числовой идентификатор разрешения на право доступа. Индентификаторы разрешения на право доступа отличаются на разных порталах. На текущий момент узнать все идентификаторы может только администратор портала в коробочной версии Битрикс24 IS_COLLAB
boolean Флаг принадлежности календаря к коллабе PERM
object Объект прав доступа текущего пользователя к календарю Название
тип Описание ALLOW
boolean Экспорт календаря разрешен Название
тип Описание view_time
boolean Просмотр времени событий календаря view_title
boolean Просмотр названия событий календаря view_full
boolean Полный доступ к информации о событии календаря add
boolean Добавление событий в календарь edit
boolean Редактирование событий в календаре edit_section
boolean Редактирование календаря access
boolean Полный доступ к календарю HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description

## ТАБЛИЦЫ

Таблица 0:
type*
string | Тип календаря:
user — календарь пользователя
group — календарь группы
company_calendar — календарь компании
location — календарь переговорной комнаты. Используется для бронирования времени в календаре переговорной комнаты через стороннее приложение
другие типы, в том числе пользовательские
ownerId*
integer | Идентификатор владельца календаря.
Для типа календаря location параметр ownerId должен иметь значение 0
