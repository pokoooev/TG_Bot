## ОПИСАНИЕ
Scope: userconsent Кто может выполнять метод: любой пользователь Метод userconsent.agreement.list возвращает список соглашений. Без параметров. Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
array Корневой элемент ответа ID
integer Идентификатор соглашения NAME
string Название соглашения ACTIVE
string Признак активности. Возможные значения: LANGUAGE_ID
string Идентификатор языка соглашения time
time Информация о времени выполнения запроса HTTP-статус: 20х, 40x, 50x Описанные ниже ошибки могут возникнуть при вызове любого метода Статус Код
