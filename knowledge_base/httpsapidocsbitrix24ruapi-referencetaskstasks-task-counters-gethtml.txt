## ОПИСАНИЕ
Scope: task Кто может выполнять метод: любой пользователь Метод tasks.task.counters.get получает значения счетчиков задач для указанного пользователя. Обязательные параметры отмечены * Название
тип Описание userId
integer Идентификатор пользователя, для которого нужно получить счетчики. По умолчанию используется текущий пользователь. Идентификатор пользователя можно получить методом получения списка пользователей groupId
integer Идентификатор группы, для задач которой нужно получилось счетчики. Передайте 0 или не указывайте параметр, чтобы учитывать все группы. Получить идентификатор можно методом создания новой группы или методом получения списка групп. type
string Роль, для которой нужно получить счетчики. Возможные роли: По умолчанию используется роль view_all Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
object Объект, в котором ключ — название счетчика, а значение — объект с описанием счетчика. Значения счетчиков: Возвращает пустой массив "result":[], если пользователь не существует или нет прав на получение счетчиков указанного пользователя total
integer На текущий момент не используется. Всегда возвращает значение 1 time
time Информация о времени выполнения запроса Название
тип Описание counter
integer Количество code
integer Внутренний код счетчика HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде Код Описание Значение 0 Invalid value {value} to match with parameter {userId}. Should be value of type int. ## ТАБЛИЦЫ

Таблица 0:
userId
integer | Идентификатор пользователя, для которого нужно получить счетчики.
По умолчанию используется текущий пользователь.
Идентификатор пользователя можно получить методом получения списка пользователей
groupId
integer | Идентификатор группы, для задач которой нужно получилось счетчики.
Передайте 0 или не указывайте параметр, чтобы учитывать все группы.
Получить идентификатор можно методом создания новой группы или методом получения списка групп.
type
string | Роль, для которой нужно получить счетчики. Возможные роли:
view_all — все роли
view_role_responsible — делаю
view_role_accomplice — помогаю
view_role_auditor — наблюдаю
view_role_originator — поручил
По умолчанию используется роль view_all

Таблица 1:
result
object | Объект, в котором ключ — название счетчика, а значение — объект с описанием счетчика.
Значения счетчиков:
new_comments — непрочитанные комментарии
expired — просроченные задачи
Возвращает пустой массив "result":[], если пользователь не существует или нет прав на получение счетчиков указанного пользователя
total
integer | На текущий момент не используется. Всегда возвращает значение 1
time
time | Информация о времени выполнения запроса

Таблица 2:
counter
integer | Количество
code
integer | Внутренний код счетчика

Таблица 3:
error
string | Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания
error_description
error_description | Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде
