# Завершить звонок
URL: https://apidocs.bitrix24.ru/api-reference/telephony/telephony-external-call-finish.html

## ОПИСАНИЕ
Scope: telephony Кто может выполнять метод: любой пользователь Метод telephony.externalcall.finish завершает звонок, фиксирует его в статистике, скрывает у пользователя карточку звонка. Параметр / Тип Описание CALL_ID*
string Идентификатор звонка из метода telephony.externalcall.register. USER_ID*
int Идентификатор пользователя. Ответственный в лиде будет изменен на переданный USER_ID. Ответственный будет меняться только для лида, созданного автоматически при вызове метода telephony.externalcall.register. Для существующего лида ответственный не меняется. DURATION*
int Длительность в сек. COST
double Стоимость звонка. COST_CURRENCY
string Валюта, в которой указана стоимость звонка. Список валют можно получить методом crm.currency.list. STATUS_CODE
string Код вызова (смотрите таблицу кодов вызова) FAILED_REASON
string Причина несостоявшегося звонка. RECORD_URL
string URL файла (желательно mp3, возможен flac) с записью звонка.
Портал осуществит две попытки скачать запись по указанному адресу. В случае неудачи, запись приложена не будет и никакого уведомления об ошибке отправлено не будет. Примечание Данный параметр является устаревшим и оставлен исключительно для обратной совместимости. Использовать его крайне не рекомендуется. Для загрузки записи звонка используйте метод telephony.externalCall.attachRecord. VOTE
int Оценка звонка пользователем (если АТС поддерживает функционал оценки разговора). ADD_TO_CHAT
int Добавлять ли сообщение о совершенном звонке сотруднику Б24 в чат (Возможные значения 0 или 1, по умолчанию 0). Обязательные параметры отмечены * Примечание Вместо USER_ID также может принимать USER_PHONE_INNER. Код Описание 200 Успешный звонок 304 Пропущенный звонок 603 Отклонено 603-S Вызов отменен 403 Запрещено 404 Неверный номер 486 Занято 484 Данное направление недоступно 503 Данное направление недоступно 480 Временно недоступен 402 Недостаточно средств на счету 423 Заблокировано OTHER Не определен Метод возвращает массив, аналогичный методу voximplant.statistic.get.
