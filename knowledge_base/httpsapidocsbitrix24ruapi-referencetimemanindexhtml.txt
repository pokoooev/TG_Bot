## ОПИСАНИЕ
Учет рабочего времени в Битрикс24 помогает организовать рабочий процесс и контролировать соблюдение графика сотрудниками. Быстрый переход: все методы и события Пользовательская документация: время и отчеты в Битрикс24 Учет рабочего времени фиксирует отработанные часы сотрудника. Для этого сотрудник отмечает начало и конец рабочего дня в системе. Управлять рабочим днем можно группой методов timeman.*. Пользовательская документация Рабочий график задает режим и продолжительность работы сотрудников. Получить настройки рабочего графика можно методом timeman.schedule.get Пользовательская документация Учет рабочего времени проверяет соответствие рабочего времени сотрудника установленному графику. Система записывает нарушения графика и руководитель может просматривать отчеты о нарушениях. Работать с отчетами и настраивать контроль времени можно с помощью группы методов timeman.timecontrol.*. Пользовательская документация Офисная сеть — это группа IP-адресов, используемых в локальной сети организации. Работа с диапазонами IP-адресов офисной сети выполняется методами группы timeman.networkrange.*. Scope: timeman Кто может выполнять метод: в зависимости от метода Метод Описание timeman.open Начать новый рабочий день или возобновить закрытый timeman.pause Поставить паузу в рабочем дне timeman.close Закрыть рабочий день timeman.status Получить информацию о текущем рабочем дне пользователя timeman.settings Получить настройки рабочего времени пользователя Метод Описание timeman.schedule.get Получает рабочий график по идентификатору Метод Описание timeman.timecontrol.report.add Отправляет отчет о выявленном отсутствии timeman.timecontrol.reports.get Получает отчет о выявленных отсутствиях timeman.timecontrol.settings.get Получает настройки инструмента контроля времени timeman.timecontrol.settings.set Устанавливает настройки инструмента контроля времени timeman.timecontrol.reports.settings.get Получает пользовательские настройки для построения интерфейса отчетов инструмента контроля времени timeman.timecontrol.reports.users.get Получает список пользователей указанного подразделения Метод Описание timeman.networkrange.get Получает диапазоны сетевых адресов, входящие в офисную сеть timeman.networkrange.set Устанавливает диапазоны сетевых адресов, входящие в офисную сеть timeman.networkrange.check Проверяет входит ли IP-адрес в диапазоны сетевых адресов офисной сети

## ТАБЛИЦЫ

Таблица 0:
timeman.open | Начать новый рабочий день или возобновить закрытый
timeman.pause | Поставить паузу в рабочем дне
timeman.close | Закрыть рабочий день
timeman.status | Получить информацию о текущем рабочем дне пользователя
timeman.settings | Получить настройки рабочего времени пользователя

Таблица 1:
timeman.schedule.get | Получает рабочий график по идентификатору

Таблица 2:
timeman.timecontrol.report.add | Отправляет отчет о выявленном отсутствии
timeman.timecontrol.reports.get | Получает отчет о выявленных отсутствиях
timeman.timecontrol.settings.get | Получает настройки инструмента контроля времени
timeman.timecontrol.settings.set | Устанавливает настройки инструмента контроля времени
timeman.timecontrol.reports.settings.get | Получает пользовательские настройки для построения интерфейса отчетов инструмента контроля времени
timeman.timecontrol.reports.users.get | Получает список пользователей указанного подразделения

Таблица 3:
timeman.networkrange.get | Получает диапазоны сетевых адресов, входящие в офисную сеть
timeman.networkrange.set | Устанавливает диапазоны сетевых адресов, входящие в офисную сеть
timeman.networkrange.check | Проверяет входит ли IP-адрес в диапазоны сетевых адресов офисной сети
