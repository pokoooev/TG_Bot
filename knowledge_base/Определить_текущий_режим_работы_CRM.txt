# Определить текущий режим работы CRM
URL: https://apidocs.bitrix24.ru/api-reference/crm/crm-settings-mode-get.html

## ОПИСАНИЕ
Название метода: crm.settings.mode.get Scope: crm Кто может выполнять метод: любой пользователь Метод возвращает текущие настройки режима работы CRM: классический режим CRM (с лидами) или простой режим CRM (без лидов). Этот режим влияет на целый ряд сценариев работы CRM и для лучшего понимания мы рекомендуем прочитать соответствующую статью пользовательской документации. Метод вызывается без параметров. Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
integer Возвращает значение, определённое в crm.enum.settings.mode time
time Информация о времени выполнения запроса Название
