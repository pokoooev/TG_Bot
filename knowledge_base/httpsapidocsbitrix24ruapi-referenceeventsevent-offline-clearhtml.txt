## ОПИСАНИЕ
Кто может выполнять метод: любой пользователь Метод event.offline.clear производит очистку записей в очереди офлайн-событий. Доступность офлайн-событий можно проверить через метод feature.get. Метод работает только в контексте авторизации приложения. Обязательные параметры отмечены * Название
тип Описание process_id*
string Идентификатор процесса, который занимается обработкой записей id
array Массив идентификаторов записей, которые нужно вычистить. По умолчанию будут вычищены все записи, помеченные переданным process_id message_id
array Массив значений поля MESSAGE_ID записей, которые нужно вычистить. Игнорируется, если указан параметр id. По умолчанию будут вычищены все записи, помеченные переданным process_id Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
boolean Успешность выполнения time
time Информация о времени выполнения запроса HTTP-статус: 20х, 40x, 50x Описанные ниже ошибки могут возникнуть при вызове любого метода Статус Код
