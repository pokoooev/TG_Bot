## ОПИСАНИЕ
Scope: task Кто может выполнять метод: любой пользователь Метод создает зависимость одной задачи от другой. Обязательные параметры отмечены * Название
тип Описание taskIdFrom*
integer Идентификатор задачи, от которой создается зависимость taskIdTo*
integer Идентификатор задачи, для которой создается зависимость linkType*
integer Тип зависимости: Идентификатор задачи можно получить при создании новой задачи или методом получения списка задач. Как использовать примеры в документации HTTP-статус: 200 Пример успешно выполненого запроса Название
тип Описание result
object В случае успеха содержит пустой массив time
time Информация о времени выполнения запроса HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде Код Описание ILLEGAL_NEW_LINK Связь между задачами уже существует ACTION_NOT_ALLOWED Невозможно добавить связь между задачами HTTP-статус: 20х, 40x, 50x Описанные ниже ошибки могут возникнуть при вызове любого метода Статус Код

## ТАБЛИЦЫ

Таблица 0:
taskIdFrom*
integer | Идентификатор задачи, от которой создается зависимость
taskIdTo*
integer | Идентификатор задачи, для которой создается зависимость
linkType*
integer | Тип зависимости:
0 — связь старт-старт
1 — связь старт-финиш
2 — связь финиш-старт
3 — связь финиш-финиш

Таблица 1:
result
object | В случае успеха содержит пустой массив
time
time | Информация о времени выполнения запроса

Таблица 2:
error
string | Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания
error_description
error_description | Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде

Таблица 3:
ILLEGAL_NEW_LINK | Связь между задачами уже существует
ACTION_NOT_ALLOWED | Невозможно добавить связь между задачами
