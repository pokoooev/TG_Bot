## ОПИСАНИЕ
Scope: landing Кто может выполнять метод: любой пользователь Метод landing.repowidget.debug включает режим отладки для всех виджетов текущего приложения. В этом случае vue-приложение будет сообщать больше ошибок в js-консоли для удобства разработчика. По умолчанию режим отключен. Обязательные параметры отмечены * Название
тип Описание enable
boolean Флаг активности режима отладки Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
boolean В случае успеха — true, в противном случае — ошибка time
time Информация о времени выполнения запроса HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде Код Описание APP_NOT_FOUND REST-приложение не найдено. ## ПРИМЕРЫ КОДА

landing.repowidget.debug
---

try
{
 const response = await $b24.callMethod(
  'landing.repowidget.debug', {
   enable: true,
  }
 );
 
 const result = response.getData().result;
 if(result.error())
  console.error(result.error());
 else
  console.info(result);
}
catch(error)
{
 console.error('Error:', error);
}
