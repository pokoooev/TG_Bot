# Получить данные прикрепленного голосования
URL: https://apidocs.bitrix24.ru/api-reference/vote/vote.attachedvote.get.html

## ОПИСАНИЕ
Scope: vote Кто может выполнять метод: пользователь с правами чтения голосования Метод vote.AttachedVote.get возвращает данные прикрепленного голосования. Доступно три варианта вызова метода. Обязательные параметры отмечены * Название
тип Описание attachId*
integer Идентификатор прикрепленного голосования, может быть равен voteId из результатов vote.Integration.Im.send Обязательные параметры отмечены * Название
тип Описание moduleId*
string Идентификатор модуля, возможные значения: entityType*
string Тип объекта, возможные значения: entityId*
integer Идентификатор элемента, возможные значения: Обязательные параметры отмечены * Название
тип Описание signedAttachId*
string Подписанный идентификатор прикрепления, получить можно методом vote.AttachedVote.get, параметр ответа signedAttachId Как использовать примеры в документации Пример вызова через идентификатор привязанного опроса. Пример вызова через связанный с опросом элемент. HTTP-статус: 200 Название
тип Описание result
object Корневой элемент ответа. Содержит объект attach с информацией о прикрепленном голосовании, структура описана ниже time
time Информация о времени выполнения запроса Название
тип Описание ID
integer Идентификатор прикрепленного голосования VOTE_ID
integer Идентификатор самого голосования COUNTER
integer Счетчик голосов QUESTIONS
array Массив с вопросами опроса ANONYMITY
integer Уровень анонимности опроса OPTIONS
integer Доступность переголосования userAnswerMap
array Карта ответов текущего пользователя canEdit
bool Может ли текущий пользователь редактировать опрос canVote
bool Может ли текущий пользователь проголосовать canRevote
bool Может ли текущий пользователь переголосовать isVoted
bool Проголосовал ли уже текущий пользователь signedAttachId
string Подписанный идентификатор resultUrl
string URL-адрес для просмотра результатов опроса downloadUrl
string URL-адрес для скачивания результатов опроса entityId
integer Идентификатор элемента, к которому прикреплен опрос isFinished
bool Завершен ли опрос HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. ## ТАБЛИЦЫ

Таблица 0:
attachId*
integer | Идентификатор прикрепленного голосования, может быть равен voteId из результатов vote.Integration.Im.send

Таблица 1:
moduleId*
string | Идентификатор модуля, возможные значения:
Im для опроса в чате,
blog для опроса в ленте
entityType*
string | Тип объекта, возможные значения:
Bitrix\\Vote\\Attachment\\ImMessageConnector для опроса в чате,
Bitrix\\Vote\\Attachment\\BlogPostConnector для опроса в ленте
entityId*
integer | Идентификатор элемента, возможные значения:
id сообщения чата с опросом, получить можно методом vote.Integration.Im.send,
id поста с опросом в ленте, получить можно методом log.blogpost.get

Таблица 2:
signedAttachId*
string | Подписанный идентификатор прикрепления, получить можно методом vote.AttachedVote.get, параметр ответа signedAttachId

Таблица 3:
result
object | Корневой элемент ответа. Содержит объект attach с информацией о прикрепленном голосовании, структура описана ниже
time
time | Информация о времени выполнения запроса

Таблица 4:
ID
integer | Идентификатор прикрепленного голосования
VOTE_ID
integer | Идентификатор самого голосования
COUNTER
integer | Счетчик голосов
QUESTIONS
array | Массив с вопросами опроса
ANONYMITY
integer | Уровень анонимности опроса
OPTIONS
integer | Доступность переголосования
userAnswerMap
array | Карта ответов текущего пользователя
canEdit
bool | Может ли текущий пользователь редактировать опрос
canVote
bool | Может ли текущий пользователь проголосовать
canRevote
bool | Может ли текущий пользователь переголосовать
isVoted
bool | Проголосовал ли уже текущий пользователь
signedAttachId
string | Подписанный идентификатор
resultUrl
string | URL-адрес для просмотра результатов опроса
downloadUrl
string | URL-адрес для скачивания результатов опроса
entityId
integer | Идентификатор элемента, к которому прикреплен опрос
isFinished
bool | Завершен ли опрос

Таблица 5:
error
string | Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания
error_description
error_description | Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде

Таблица 6:
ATTACH_NOT_FOUND | Голосование не найдено
ATTACH_READ_ACCESS_DENIED | Нет прав для участия в голосовании
