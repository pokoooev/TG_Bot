# Получить список чат-ботов
URL: https://apidocs.bitrix24.ru/api-reference/chat-bots/imbot-bot-list.html

## ОПИСАНИЕ
Scope: imbot Кто может выполнять метод: любой пользователь Метод imbot.bot.list получает список доступных чат-ботов. Без параметров. Примечание В примере используется функция restCommand. Это метод отправки данных в Битрикс24. Другой пример использования метода вы найдёте в примере ЭхоБота. Вы можете использовать для вызова метода свою функцию, javascript-метод BX24.callMethod или bitrix24-php-sdk. Как использовать примеры в документации Массив массивов, содержащий данные по чат-ботам:

## ПРИМЕРЫ КОДА

$result = restCommand(
    'imbot.bot.list',
    Array(),
    $_REQUEST[
        "auth"
    ]
);
