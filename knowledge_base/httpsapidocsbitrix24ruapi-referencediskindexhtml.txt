## ОПИСАНИЕ
Диск — это инструмент для работы с файлами. Он позволяет: Диск интегрирован с Задачами, Чатами, CRM, Рабочими группами и другими инструментами Битрикс24. Это упрощает работу за счет быстрого доступа к файлам, совместной работы над ними и автоматизации документооборота. Быстрый переход: все методы Пользовательская документация: Битрикс24 Диск Есть три вида хранилищ: Хранилищами можно управлять с помощью группы методов disk.storage.*. Пользовательская документация Папки и файлы можно создавать, перемещать, удалять, а также изменять права доступа к ним. Для этого используйте группы методов disk.folder.* и disk.file.*. Узнать версию файла можно с помощью метода disk.version.get. Пользовательская документация CRM. Файлы можно прикрепить к делам и коммерческим предложениям. За работу с делами отвечает группа методов crm.activity.*, за работу с предложениями — crm.quote.*. Бизнес-процессы. Можно запустить для документов общего диска. Управление бизнес-процессами выполняется методами bizproc.workflow.*. Задачи. Файлы прикрепляются к описанию задачи. Все участники задачи могут просматривать, редактировать и скачивать прикрепленные файлы. Работать с задачами необходимо через методы tasks.task.*. Календарь. Файлы добавляются к событию и становятся доступными всем участникам. Создать и изменить событие можно с помощью методов calendar.event.*. Лента новостей. Файлы прикрепляются к сообщениям и комментариям. Метод log.blogpost.add создает новое сообщение, а метод log.blogcomment.add — новый комментарий. Почта. Вложения из писем сохраняются на Диске. С письмами можно работать только через интерфейс Битрикс24. Если письмо сохранено в CRM, то с вложениями можно взаимодействовать методами crm.activity.* через поле FILES. Рабочие группы и проекты. Диск интегрирован в рабочие группы и проекты, для каждого из них есть отдельное хранилище. Универсальные списки. Элементы списка связаны с Диском через поле типа Файл (Диск). Создать и изменить элемент можно с помощью методов lists.element.*. Чаты. Пользователи могут обмениваться документами, фото, видео и аудио. Файлы можно просматривать и скачивать, а документы — редактировать в чате без скачивания. Прикрепленные файлы доступны всем участникам чата. Метод im.message.add отправляет сообщение с файлом в чат.
