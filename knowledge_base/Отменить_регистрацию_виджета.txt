# Отменить регистрацию виджета
URL: https://apidocs.bitrix24.ru/api-reference/widgets/placement-unbind.html

## ОПИСАНИЕ
Scope: в зависимости от места встройки Кто может выполнять метод: администратор Метод удаляет зарегистрированный обработчик места встраивания и возвращает количество удаленных обработчиков. Параметр Описание С версии PLACEMENT* Идентификатор места встраивания. HANDLER URL обработчика места встраивания. Если не указывать URL обработчика, будут удалены все обработчики указанного места встраивания, зарегистрированные приложением. * - Обязательный параметр. Как использовать примеры в документации
