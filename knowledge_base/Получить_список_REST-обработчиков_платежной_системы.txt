# Получить список REST-обработчиков платежной системы
URL: https://apidocs.bitrix24.ru/api-reference/pay-system/sale-pay-system-handler-list.html

## ОПИСАНИЕ
Scope: pay_system Кто может выполнять метод: администратор CRM (право «Разрешить изменять настройки») Метод возвращает список REST-обработчиков платежной системы. Без параметров. Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
sale_paysystem_handler[] Список зарегистрированных REST-обработчиков платежных систем time
time Информация о времени выполнения запроса HTTP-статус: 403 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. Описание не предназначено для показа конечному пользователю в необработанном виде Код Описание Статус ACCESS_DENIED Доступ запрещен. Приложение пытается изменить обработчик, добавленный другим приложением, либо недостаточно прав для обновления обработчика 403 HTTP-статус: 20х, 40x, 50x Описанные ниже ошибки могут возникнуть при вызове любого метода Статус Код

Too many requests Превышен лимит на интенсивность запросов 405 ERROR_BATCH_METHOD_NOT_ALLOWED
