# Включить/выключить публичную ссылку на документ
URL: https://apidocs.bitrix24.ru/api-reference/document-generator/document-generator-document-enable-public-url.html

## ОПИСАНИЕ
Scope: documentgenerator Кто может выполнять метод: любой пользователь Метод documentgenerator.document.enablepublicurl включает/выключает публичную ссылку на документ. Параметр Описание id Идентификатор документа. status 1 (включить), 0 (выключить) публичную ссылку на документ. 200 OK

## ПРИМЕРЫ КОДА

documentgenerator.document.enablepublicurl
---

"publicUrl": "" // публичная ссылка
---

## ТАБЛИЦЫ

Таблица 0:
id | Идентификатор документа.
status | 1 (включить), 0 (выключить) публичную ссылку на документ.
