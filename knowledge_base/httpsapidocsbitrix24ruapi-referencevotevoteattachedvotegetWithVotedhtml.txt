## ОПИСАНИЕ
Scope: vote Кто может выполнять метод: любой пользователь с правами чтения голосования Метод vote.AttachedVote.getWithVoted возвращает данные прикрепленного голосования вместе с информацией о проголосовавших пользователях. Доступно три варианта вызова метода. Обязательные параметры отмечены * Название

integer Идентификатор прикрепленного голосования, получить можно методами vote.AttachedVote.get или vote.AttachedVote.getMany pageSize
integer Количество записей на странице для списка проголосовавших.
Значение по умолчанию: 10 userForMobileFormat
boolean Формат данных пользователей для мобильных устройств.
Значение по умолчанию: false Обязательные параметры отмечены * Название
