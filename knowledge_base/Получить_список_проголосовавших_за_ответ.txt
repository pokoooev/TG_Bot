# Получить список проголосовавших за ответ
URL: https://apidocs.bitrix24.ru/api-reference/vote/vote.attachedvote.getAnswerVoted.html

## ОПИСАНИЕ
Scope: vote Кто может выполнять метод: пользователь с правами чтения голосования Метод vote.AttachedVote.getAnswerVoted возвращает список пользователей, проголосовавших за указанный вариант ответа. Доступно три варианта вызова метода. Обязательные параметры отмечены * Название
тип Описание attachId*
integer Идентификатор прикрепленного голосования, получить можно методами vote.AttachedVote.get или vote.AttachedVote.getMany answerId*
integer Идентификатор ответа, получить можно методами vote.AttachedVote.get или vote.AttachedVote.getMany pageNavigation
object Параметры постраничной навигации, например userForMobileFormat
boolean Формат данных пользователей для мобильных устройств. Значение по умолчанию: false Обязательные параметры отмечены * Название
тип Описание moduleId*
string Идентификатор модуля, возможные значения: entityType*
string Тип объекта, возможные значения: entityId*
integer Идентификатор элемента, возможные значения: answerId*
integer Идентификатор ответа, получить можно методами vote.AttachedVote.get или vote.AttachedVote.getMany pageNavigation
object Параметры постраничной навигации, например userForMobileFormat
boolean Формат данных пользователей для мобильных устройств. Значение по умолчанию: false Обязательные параметры отмечены * Название
тип Описание signedAttachId*
string Подписанный идентификатор прикрепления, получить можно методом vote.AttachedVote.get, параметр ответа signedAttachId answerId*
integer Идентификатор ответа, получить можно методами vote.AttachedVote.get или vote.AttachedVote.getMany pageNavigation
object Параметры постраничной навигации, например userForMobileFormat
boolean Формат данных пользователей для мобильных устройств. Значение по умолчанию: false Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
object Корневой элемент ответа. Содержит массив данных о проголосовавших пользователях, структура описана ниже total
