## ОПИСАНИЕ
Scope: task Кто может выполнять метод: постановщик задачи или пользователь с доступом к редактированию задачи Метод tasks.task.files.attach добавляет файл с Диска в задачу. У пользователя должен быть доступ к файлу на чтение или выше. Обязательные параметры отмечены * Название
тип Описание taskId*
integer Идентификатор задачи, к которой нужно прикрепить файл. Идентификатор задачи можно получить при создании новой задачи или методом получения списка задач fileId*
integer Идентификатор файла на Диске. Получить идентификатор файла можно двумя способами. Использовать один из методов загрузки файла: Использовать один из методов получения списка файлов: Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
object Корневой элемент ответа. Содержит объект с описанием прикрепленного файла attachmentId
integer Идентификатор прикрепления файла к задаче. Получить данные о файле по идентификатору прикрепления можно методом disk.attachedObject.get time
time Информация о времени выполнения запроса HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки. ## ТАБЛИЦЫ

Таблица 0:
taskId*
integer | Идентификатор задачи, к которой нужно прикрепить файл.
Идентификатор задачи можно получить при создании новой задачи или методом получения списка задач
fileId*
integer | Идентификатор файла на Диске.
Получить идентификатор файла можно двумя способами.
Использовать один из методов загрузки файла:
disk.storage.uploadfile
disk.folder.uploadfile
Использовать один из методов получения списка файлов:
disk.storage.getchildren
disk.folder.getchildren
