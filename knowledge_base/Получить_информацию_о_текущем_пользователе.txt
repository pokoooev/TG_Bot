# Получить информацию о текущем пользователе
URL: https://apidocs.bitrix24.ru/api-reference/user/user-current.html

## ОПИСАНИЕ
Scope: user, user_brief, user_basic Кто может выполнять метод: любой пользователь Метод user.current получает информации о текущем пользователе. Перечень полей пользователей Битрикс24, который будет получен в результате выполнения метода, зависит от скоупа приложения/вебхука. Подробности о доступе к данным пользователей можно узнать в статье. Метод параметров не имеет. Однако, сделав REST-запрос с использованием данных из $_REQUEST к домену DOMAIN и добавив AUTH_ID к запросу для доступа к Битрикс24, можно узнать какой пользователь открыл страницу в контексте Битрикс24. Как использовать примеры в документации HTTP-статус: 200 Название
