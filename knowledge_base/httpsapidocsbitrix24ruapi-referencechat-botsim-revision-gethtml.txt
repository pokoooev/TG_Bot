## ОПИСАНИЕ
Scope: im Кто может выполнять метод: любой пользователь Метод im.revision.get получает информацию о ревизиях API. Как использовать примеры в документации

## ПРИМЕРЫ КОДА

try
{
 const response = await $b24.callMethod(
  'im.revision.get',
  {}
 );
 
 const result = response.getData().result;
 console.log(result);
}
catch( error )
{
 console.error('Error:', error);
}
