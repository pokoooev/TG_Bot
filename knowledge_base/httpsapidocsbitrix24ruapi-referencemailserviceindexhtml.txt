## ОПИСАНИЕ
В Битрикс24 можно подключить почтовые ящики от сервисов Яндекс, Mail.ru, Gmail и других. Это упрощает работу с письмами. Быстрый переход: все методы С помощью REST API можно управлять только почтовыми сервисами. Подключать почтовые ящики, отправлять и получать письма нужно через интерфейс Битрикс24. Пользовательская документация Scope: mailservice Кто может выполнять метод: любой пользователь Метод Описание mailservice.fields Возвращает описание полей почтового сервиса mailservice.list Возвращает список всех почтовых сервисов mailservice.get Возвращает параметры указанного почтового сервиса mailservice.add Добавляет почтовый сервис mailservice.update Обновляет параметры почтового сервиса mailservice.delete Удаляет почтовый сервис
