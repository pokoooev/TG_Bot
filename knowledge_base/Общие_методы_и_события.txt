# Общие методы и события
URL: https://apidocs.bitrix24.ru/api-reference/common/index.html

## ОПИСАНИЕ
Несколько общих методов, доступных приложению независимо от запрашиваемых приложением разрешений: Метод Описание scope Показ разрешений. app.info Показ информации о приложении. access.name Получение названий прав доступа. server.time Возвращает текущее время сервера в формате ATOM. feature.get Проверка доступности функционала на портале. method.get Проверка доступности конкретного метода. Метод Описание user.admin Определяет обладает ли текущий пользователь правами на управление настройками приложений. user.access Определяет обладает ли текущий пользователь хотя бы одним из заданного параметром ACCESS набора прав. Метод Описание app.option.set Задать свои настройки приложения для хранения в Битрикс24. app.option.get Получить сохраненные настройки приложения из Битрикс24. user.option.set Задать свои настройки текущего пользователя для хранения в Битрикс24. user.option.get Получить сохраненные настройки текущего пользователя из Битрикс24. Событие Описание OnAppInstall Вызывается после успешной установки приложения на конкретном эккаунте Битрикс24. OnAppUninstall Вызывается после удаления приложения в конкретном эккаунте Битрикс24. OnAppMethodConfirm Вызывается после разрешения администратором портала на применение особого класса методов. onAppPayment Вызывается при оплате приложения. onAppUserAdd Вызывается при добавлении пользователя в Битрикс24.
