## ОПИСАНИЕ
Scope: calendar Кто может выполнять метод: любой пользователь Метод удаляет календарь. Обязательные параметры отмечены * Название
тип Описание type*
string Тип календаря: ownerId*
integer Идентификатор владельца календаря id*
integer Идентификатор календаря Как использовать примеры в документации HTTP-статус: 200 Название
тип Описание result
boolean Результат удаления календаря. Возвращает true, если календарь успешно удален HTTP-статус: 400 Название
тип Описание error
string Строковый код ошибки. Может состоять из цифр, латинских букв и знака подчеркивания error_description
error_description Текстовое описание ошибки.
