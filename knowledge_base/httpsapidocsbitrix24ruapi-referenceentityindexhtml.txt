## ОПИСАНИЕ
Хранилища позволяют разработчикам создавать приложения, которые расширяют возможности Битрикс24. В них можно хранить разнообразную информацию: записи о клиентах, сведения об инвентаризации, информацию о товарах и другие данные. Каждое хранилище представляет собой информационный блок и доступно после регистрации приложения в Битрикс24. Визуального интерфейса хранилища не имеют. Хранилищами приложений можно управлять с помощью группы методов entity.*. Быстрый переход: все методы Разделы. Предназначены для группировки данных и построения удобной иерархии. Разделами можно управлять с помощью методов entity.section.*. Элементы. Хранят данные приложений. Создавать, изменять и удалять элементы можно с помощью методов entity.item.*. Любое хранилище позволяет настроить дополнительные свойства элементов. Чтобы управлять ими, используйте методы entity.item.property.*. Scope: entity Кто может выполнять метод: любой пользователь Метод Описание entity.add Создает хранилище данных entity.update Изменяет параметры хранилища данных entity.rights Получает или изменяет прав доступа к хранилищу entity.get Получает параметры хранилища или список всех хранилищ приложения entity.delete Удаляет хранилище Метод Описание entity.section.get Получает список разделов хранилища entity.section.add Добавляет раздел хранилища entity.section.update Изменяет раздел хранилища entity.section.delete Удаляет раздел хранилища Метод Описание entity.item.get Получает список элементов хранилища entity.item.add Добавляет элемент хранилища entity.item.update Изменяет элемент хранилища entity.item.delete Удаляет элемент хранилища Метод Описание entity.item.property.get Получает список дополнительных свойств элементов хранилища entity.item.property.add Добавляет дополнительное свойство элементов хранилища entity.item.property.update Изменяет дополнительное свойство элементов хранилища entity.item.property.delete Удаляет дополнительное свойство элементов хранилища

## ТАБЛИЦЫ

Таблица 0:
entity.add | Создает хранилище данных
entity.update | Изменяет параметры хранилища данных
entity.rights | Получает или изменяет прав доступа к хранилищу
entity.get | Получает параметры хранилища или список всех хранилищ приложения
entity.delete | Удаляет хранилище

entity.section.get | Получает список разделов хранилища
entity.section.add | Добавляет раздел хранилища

entity.section.delete | Удаляет раздел хранилища

Таблица 2:
entity.item.get | Получает список элементов хранилища
entity.item.add | Добавляет элемент хранилища
entity.item.update | Изменяет элемент хранилища
entity.item.delete | Удаляет элемент хранилища

entity.item.property.delete | Удаляет дополнительное свойство элементов хранилища
