## ОПИСАНИЕ
В календарь можно встроить приложение. В верхней части календаря в списке видов отображения есть место для встройки CALENDAR_GRIDVIEW, куда можно добавить свой пункт. Подробнее о виджете — в статье Виджет в календаре. Исспользуйте метод placement.bind, чтобы связать приложение с календарем. Параметры PLACEMENT, HANDLER и TITLE определяют, где и как будет отображаться ваше приложение. Если в приложении вывести все параметры, которые передаются в запросе к приложению: Можно увидеть, что передаются определенные параметры, например диапазон дат, который отображается в календаре: Эти параметры можно использовать для настройки отображения вашего приложения. Метод getEvents получает события календаря. Метод viewEvent открывает карточку для просмотра события. Метод addEvent открывает карточку для добавления нового события. Метод editEvent открывает карточку для редактирования события. Метод deleteEvent удаляет событие. События, которые можно отслеживать в месте встройки: Событие Описание Calendar.customView:refreshEntries Обновление событий Calendar.customView:decreaseViewRangeDate Нажатие на стрелочку назад, то есть открытие календаря за предыдущие даты Calendar.customView:increaseViewRangeDate Нажатие на стрелочку вперед, то есть открытие календаря на следующие даты Calendar.customView:adjustToDate Переход к конкретной дате

## ПРИМЕРЫ КОДА

try
{
 const response = await $b24.callMethod(
  'placement.bind',
  {
   PLACEMENT: 'CALENDAR_GRIDVIEW',
   HANDLER: 'http://your_site/handler.php',
   TITLE: 'Custom tab'
  }
 );
 
 const result = response.getData().result;
 console.log(result);
}
catch( error )
{
 console.error('Error:', error);
}

try
{
 const dateFrom = new Date();
 const dateTo = new Date(dateFrom.getTime() + 86400 * 30 * 1000); // Умножаем на 1000, чтобы преобразовать секунды в миллисекунды
 dateFrom.setHours(0, 0, 0, 0);
 dateTo.setHours(0, 0, 0, 0);

 const response = await $b24.callMethod(
  'placement.getEvents',
  {
   dateFrom: dateFrom,
   dateTo: dateTo
  }
 );

 const events = response.getData().result;
 console.log('getEvents response:');
 console.dir(events);
}
catch( error )
{
 console.error('Error:', error);
}

## ТАБЛИЦЫ

Таблица 0:
Calendar.customView:refreshEntries | Обновление событий
Calendar.customView:decreaseViewRangeDate | Нажатие на стрелочку назад, то есть открытие календаря за предыдущие даты
Calendar.customView:increaseViewRangeDate | Нажатие на стрелочку вперед, то есть открытие календаря на следующие даты
Calendar.customView:adjustToDate | Переход к конкретной дате
